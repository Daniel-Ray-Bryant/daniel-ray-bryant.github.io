C++ Projects

Project Title 1: Starwood Hollow

A small-scale 2D game built from scratch using C++. This is a simple sidescroller where you play as an alien exploring a mysterious planet, fighting off evil skeletons and avoiding dangerous hazards on your way!

My Contributions:

  • Developed a custom sidescroller/platformer controller.
  • Created a procedural level flow for generating diverse and engaging levels.
  • Engineered a basic physics simulation for gravity and collision detection.
  • Implemented enemy AI with simple state machines for patrol and various attack behaviors.
  • Designed a health and damage system for both the player and enemies.
  • Added sound effects and background music for an immersive experience.

Project Title 2: Axe Game

A simple C++ game where the player must attempt to cross a line as many times as possible without being hit by the axe, where each pass of the line increases the speed at which the axe moves.

My Contributions:

  • Created a point system along with high-score tracking between sessions.
  • Implemented a simple AI for the axe's movement pattern.
  • Developed a simple player control system for responsive control.
  • Designed a basic collision detection system to determine hits and misses.
  • Added sound effects for player actions and game events.
